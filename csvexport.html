CSV Export¶

The export_sms_csv.sh script will SSH into a SMS hosting box to dump an export of the current state of the SMS to a CSV file which is then ingested by this tool. The CSV consists of a single header line and then rows of data. The CSV columns are not defined here - rather they are defined on the MediaItem in the API Reference.
